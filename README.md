# Lab-4
Программа запрашивает у пользователя кол-во сотрудников, расстояние от работы до дома и тарифы за проезд одного километра в такси.
Программа выводит число — номер такси, в которое должен сесть первый сотрудник, второе число — номер такси, в которое должен сесть второй и т.д., так чтобы суммарные затраты на такси были минимальны.
